# Contributing to SecurOS

Thank you for your interest in contributing to SecurOS.

---

## Ways to Contribute

- Bug reports
- Documentation improvements
- Testing and feedback
- Build system enhancements

---

## Contribution Guidelines

- Follow Arch Linux and ArchISO best practices
- Keep changes minimal and well-documented
- Test changes before submitting pull requests

---

## Submitting Issues

Submit issues via GitHub Issues under the Secur-OS organization. Provide logs, steps to reproduce, and system details when possible.
